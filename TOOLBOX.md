# 🧰 The AI Creator's Toolbox

## 🤗 The Engineer's Home Base — Hugging Face

**[Hugging Face](https://huggingface.co)** 🆓/💰 — if AI engineers had a town square, this is it. It hosts:
- **Models** — hundreds of thousands of open models you can download and run (LLMs, image, audio, vision).
- **Datasets** — ready-to-use training/eval data.
- **Spaces** — free hosting to deploy ML demos (Gradio/Streamlit). *Put your portfolio projects here.*
- **Libraries** — `transformers`, `diffusers`, `datasets` — the backbone of practical AI work.

> If you learn *one* platform deeply on this whole roadmap, make it Hugging Face.

## 🧭 How to actually use this list

1. **Don't install everything.** Pick one per category as you reach the matching level.
2. **Free-first.** Almost every category has a free tier strong enough to learn and build a portfolio piece.
3. **The meta-skill** is *evaluating* tools fast: try it for 20 minutes on a real task, keep it or drop it. That instinct is what makes you "unstoppable" as the landscape keeps shifting.
